## 安装git

* ubantu上安装git及使用

1.sudo apt-get install git.

2.进行简单配置

  git config --global user.name "Your Name"
  
  git config --global user.email "email@example.com"
  
3.生成秘钥

  ssh-keygen -t rsa -C "youremail@example.com"
  
4.找到公钥，复制到github

touch readme.txt
vi readme.txt
git add readme.txt
git commit -m “描述”
